# IPL Win Predictor 

An end-to-end Machine Learning project that predicts the probability of winning an IPL match based on live match inputs such as score, overs, wickets, current run rate, and required run rate.

##  Project Highlights
- Achieved 82% model accuracy
- Performed data preprocessing and feature engineering
- Built and evaluated classification models using Scikit-learn
- Deployed as an interactive web application using Streamlit

##  Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit

##  Features Used
- Batting Team
- Bowling Team
- Host City
- Target
- Current Score
- Overs Completed
- Wickets Out
- Current Run Rate
- Required Run Rate
